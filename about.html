About this site

This is a static site hosted on GitHub Pages. There is no application server. Country pages and index data are generated from structured source records and updated automatically via GitHub Actions.

What the map shows

The map uses color-coded markers to reflect country-level persecution severity: red for Extreme/Very High, orange for High, yellow for Moderate, and blue for Low. Clicking a country opens a page with context, documented incidents, and linked sources.

Data sources

This project relies on publicly available reports. Country pages reference the specific reports used for each entry. The table below lists the organizations whose data feeds into this site.

OK — source fetched successfully Partial — partial data retrieved Error — fetch failed Skipped — not yet fetched

Update workflow

A scheduled GitHub Actions workflow regenerates the site data from source records. When data changes are committed, the workflow rebuilds country pages, search assets, and map data files, then publishes the updated site.

Privacy and access

This site does not collect personal data. Map tiles and search indexing are loaded from public libraries. If you use an ad blocker, the Leaflet tile and search layers may be affected.

Attribution note: This project is unaffiliated with any government, NGO, or religious organization listed as a source. Source information remains the property of each provider and is subject to their terms and attribution requirements.

About the creator

This site was created by Joey as a companion tool for SIT — Solidarietà Internazionale Trinitaria, the Trinitarian Order’s international solidarity ministry supporting persecuted Christians worldwide. SIT works to raise awareness and provide concrete aid to Christians facing persecution because of their faith.

Contributing

Edits are tracked in the repository as structured data. If you want to suggest a source, correction, or new country entry, open an issue or pull request in the project repository.
